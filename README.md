# Домашнє завдання: useAsync Hook

Цей проект демонструє використання кастомного хука `useAsync` для роботи з асинхронними даними в React.

## Що реалізовано

- ✅ Кастомний хук `useAsync` для керування асинхронними операціями
- ✅ Компонент `MessageComponent` що використовує цей хук
- ✅ Обробка станів завантаження, успіху та помилок
- ✅ Простий та зрозумілий інтерфейс
- ✅ Симуляція мережевих запитів з випадковими помилками

## Структура проекту

```
src/
├── hooks/
│   └── useAsync.js          # Кастомний хук
├── components/
│   └── MessageComponent.jsx # Основний компонент
├── App.jsx                  # Головний компонент додатка
└── App.css                  # Стилі
```

## Як запустити

1. Встановіть залежності:

```bash
npm install
```

2. Запустіть проект у режимі розробки:

```bash
npm run dev
```

3. Відкрийте браузер за адресою http://localhost:5173

## Функціональність

- Натисніть кнопку "Отримати повідомлення"
